# TASK-1
This project analyzes the Student Performance Dataset (student-mat.csv) using Python, Pandas, Seaborn, and Matplotlib.
The goal is to explore student grades, study habits, and demographic factors to understand what influences academic performance.

Project Overview:
- Loaded and explored the dataset
- Cleaned the data
- Checked for missing values
- Removed duplicates
- Inspected shape and data types
- Performed statistical analysis
- Visualized key insights
- Documented each step using Markdown in Jupyter Notebook

Key Analysis Performed
- Average Final Grade (G3)
- Students Scoring Above 15
- Correlation Analysis
- Gender Comparison
- Data Visualization

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
